# Python Calculator

A simple command-line calculator written in Python.
This project is designed as a clean, beginner-friendly example of how to structure and run a small Python program.

## ‚ú® What this project does

The calculator allows you to perform basic arithmetic operations:

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)

It also includes basic error handling, such as protection against division by zero.

## üéØ Why this project exists

This repository is useful if you:

* Are learning Python basics
* Want a minimal example of a CLI (command-line interface) program
* Need a simple reference project for functions and program flow
* Want a small project to practice Git/GitHub workflow

## üìÇ Project structure

```
calculator.py
```

The entire logic is contained in a single file for simplicity and easy reuse.

## ‚ñ∂Ô∏è How to run

Make sure you have Python 3 installed.

```bash
python calculator.py
```

You will be prompted to:

1. Choose an operation (`+ - * /`)
2. Enter the first number
3. Enter the second number

The result will be printed directly to the console.

## üß† Example

```

Available operations: +  -  *  /
Operation: *
First number: 6
Second number: 7
Result: 42
```
